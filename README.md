# android_device_oppo_A33
TWRP Device Tree For oppo A33
上市时间：2015年10月  操作系统：ColorOS 2.1（基于 Android 5.1）
处理器与存储 CPU型号：高通 骁龙410（MSM8916）  CPU频率：1.2GHz 四核  GPU：Adreno 306  RAM容量：2GB  ROM容量：16GB  存储扩展：支持MicroSD卡，最高128GB 
显示屏 屏幕尺寸：5英寸  屏幕材质：TFT  分辨率：960 x 540像素 
摄像头 后置摄像头：800万像素  前置摄像头：500万像素  视频拍摄：支持1080p（1920×1080）视频录制 
电池与充电 电池容量：2400mAh（不可拆卸） 充电接口：Micro USB v2.0  耳机接口：3.5mm 
网络与连接 网络制式：全网通（支持移动、联通、电信4G/3G/2G） SIM卡类型：双卡（Micro SIM卡 / Nano SIM卡） 无线连接：Wi-Fi (802.11 b/g/n)，蓝牙4.0 
尺寸与重量 手机尺寸：142.7 x 71.7 x 7.55 mm  手机重量：146g
